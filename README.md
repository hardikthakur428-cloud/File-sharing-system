A simple web-based file sharing system built using Python (Flask) that allows users to upload, view, and download files easily.

🚀 Features

Upload files to server

View list of uploaded files

Download files anytime

Simple and clean interface

🛠️ Technologies Used

Backend: Python (Flask)

Frontend: HTML

Storage: Local file system

📂 Project Structure
file-sharing-system/
│── app.py
│── uploads/
│── templates/
    │── index.html
⚙️ Installation & Setup

Clone the repository

git clone https://github.com/your-username/file-sharing-system.git
cd file-sharing-system

Install dependencies

pip install flask

Run the application

python app.py

Open in browser

http://127.0.0.1:5000/
📌 How It Works

Open the web application

Upload a file using the upload form

File gets stored in the uploads folder

Uploaded files are displayed on the homepage

Click download to retrieve any file

🔒 Limitations

No user authentication

No file size validation

Basic security implementation
